# Geometrias_j70

Generador paramétrico (Python) de geometrías 3D de velas J/70 (mayor y foque) listas para OpenFOAM/snappyHexMesh.

- Especificación: `docs/SPEC.md`
- Fuentes reglas (copiadas desde el workspace del magíster): `docs/REFERENCES.md`

> Objetivo: producir superficies trianguladas (STL/OBJ) **reproducibles**, con **parametrización consistente** (vector de parámetros) orientada a generación masiva de casos para CFD + entrenamiento de redes.
